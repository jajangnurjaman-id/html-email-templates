# Feedback Email Template

Responsive HTML email template for requesting feedback from users.

## Preview
![Preview](feedback-email/preview.png)

## Features

- Responsive layout
- Clean design
- Basic HTML and CSS
- Compatible with most email clients
- Easy to customize

## Usage

1. Download the template
2. Replace "Your Logo"
3. Update feedback message
4. Send using your email service
