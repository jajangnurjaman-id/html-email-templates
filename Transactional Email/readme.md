# Transactional Email Template

Responsive HTML email template for order confirmations, receipts, and notifications.

## Preview
![Preview](transactional-email/preview.png)

## Features

- Responsive layout
- Clean design
- Basic HTML and CSS
- Compatible with most email clients
- Easy to customize

## Usage

1. Download the template
2. Replace "Your Logo"
3. Update transaction details
4. Send using your email service
